Zmap: Single-cell spatial mapping by diverse space zoning plans（ZMap）
=========================================================================
.. raw:: html

**Zmap** is a robust approach that probabilistically maps  dissociated single cells from an scRNAseq dataset to their spatial origin.

GitHub (currently recomended)
---------------

First, clone the repository using ``git``::

    git clone https://github.com/caiquanyou/Zmap

Then, ``cd`` to the sagenet folder and run the install command::

    cd Zmap
    python setup.py install #or pip install .
    
PyPI
--------

The easiest way to get Zmap is through pip using the following command::

    pip install Zmap
