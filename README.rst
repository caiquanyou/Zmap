Zmap: Single-cell spatial mapping by diverse space zoning plans（ZMap）
=========================================================================

Introduction
------------
Zmap is a robust approach for single-cell spatial mapping by diverse space zoning plans (ZMap). This tool probabilistically maps dissociated single cells from an scRNAseq dataset to their spatial origin, offering a sophisticated method for spatially resolving cellular data.

Installation
------------
Zmap can be installed either through GitHub or PyPI.

To install from GitHub:
.. code-block:: bash

    git clone https://github.com/caiquanyou/Zmap
    cd Zmap
    python setup.py install # or pip install .

Alternatively, install via PyPI using:
.. code-block:: bash

    pip install Zmap

Usage
-----
After installation, Zmap can be used in Python as follows:
.. code-block:: python

    import Zmap as zm
